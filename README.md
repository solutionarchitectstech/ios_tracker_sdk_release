# SDK installation

## SPM

Add our SDK as a dependency to your project, like:

```swift
.package(url: "https://github.com/solutionarchitectstech/ios_tracker_sdk_release", exact: "1.6.0")
```

## Cocoapods

```
pod 'SATrackingSDKStandard', :git => 'https://github.com/solutionarchitectstech/ios_tracker_sdk_release.git', :tag => '1.6.0'
```
